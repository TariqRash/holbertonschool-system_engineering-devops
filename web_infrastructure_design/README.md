# Web Infrastructure Design

This project covers the design of web infrastructure from a simple single-server setup to a fully scaled, secured, and monitored multi-server architecture.

## Tasks

- **0-simple_web_stack** — Single server LAMP stack hosting www.foobar.com
- **1-distributed_web_infrastructure** — Three-server setup with HAproxy load balancer and Primary-Replica MySQL
- **2-secured_and_monitored_web_infrastructure** — Secured with firewalls, SSL, and monitoring clients
- **3-scale_up** — Split components onto dedicated servers with clustered load balancer
